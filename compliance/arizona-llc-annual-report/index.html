Arizona LLC Annual Report: Deadline, Cost, and Filing Guide (2026)

Last updated: February 2026 · Applies to: Arizona LLCs

Arizona requires every LLC to file an annual report with the Arizona Corporation Commission. Like Ohio, Arizona does not charge a filing fee — the annual report is free. The report confirms your LLC's basic information and keeps it in good standing.

When is it due?

The Arizona annual report is due within a 60-day window surrounding the anniversary of your LLC's approval date. You can file starting 60 days before and must file no later than 60 days after.

How much does it cost?

There is no filing fee for the Arizona LLC annual report. It is completely free to file online.

How to file

Filing steps

Go to the Arizona Corporation Commission eCorp portal
Search for your LLC by name or file number
Select the annual report filing option
Confirm or update your LLC information
Submit — no payment required
Save your confirmation

What happens if you don't file?

If you fail to file the annual report within the 60-day window, the Arizona Corporation Commission may administratively dissolve your LLC. You can apply for reinstatement by filing the delinquent report(s) and paying a reinstatement fee.

Do you need a registered agent in Arizona?

Yes. Arizona requires every LLC to maintain a statutory agent (Arizona's term for registered agent) with a physical Arizona street address.

Need a registered agent in Arizona?

A professional registered agent ensures you receive all legal and state documents on time and keeps your LLC in good standing.

Arizona LLC compliance checklist

Ongoing requirements for Arizona LLCs

File annual report within 60 days of your anniversary date ($0 — free)
Maintain a statutory agent with an Arizona street address
File Arizona income tax return (Form 165 for partnerships/multi-member LLCs)
File federal tax return (Form 1065 or Schedule C)
Collect and remit Arizona Transaction Privilege Tax if applicable

Frequently asked questions

Is the Arizona annual report really free?

Yes. Arizona does not charge any fee. However, you still need to file it — failure to file results in dissolution.

What is a statutory agent?

A statutory agent is Arizona's term for a registered agent — a person or company that receives legal documents on your behalf.

Can I file by mail?

Arizona encourages electronic filing through the eCorp portal. Paper filing is available but slower.
