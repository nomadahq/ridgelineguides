Maine LLC Annual Report: Deadline, Cost, and Filing Guide (2026)

Last updated: February 2026 · Applies to: Maine LLCs

Maine LLCs must file an annual report by June 1 each year. The filing fee is $85.

When is it due?

The annual report is due June 1 each year for all Maine LLCs.

How much does it cost?

The annual report fee is $85.

How to file

Filing steps

Go to Maine SOS Corporate Division
Search for your LLC
File the annual report
Pay the $85 fee
Save confirmation

What happens if you miss the deadline?

Failure to file leads to revocation of LLC status. Reinstatement is possible but requires overdue filings and fees.

Do you need a registered agent in Maine?

Yes. Maine requires all LLCs to maintain a registered agent with a Maine address.

Need a registered agent in Maine?

A registered agent keeps your LLC in good standing and ensures you receive all legal and state documents on time.

Frequently asked questions

Is the deadline always June 1?

Yes, for all Maine LLCs regardless of formation date.

Can I file online?

Yes. File through the Maine Secretary of State website.

What if I miss the deadline?

Your LLC risks revocation. File as soon as possible to avoid losing your LLC status.
