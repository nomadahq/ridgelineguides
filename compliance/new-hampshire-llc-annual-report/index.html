New Hampshire LLC Annual Report: Deadline, Cost, and Filing Guide (2026)

Last updated: February 2026 · Applies to: New Hampshire LLCs

New Hampshire LLCs must file an annual report by April 1 each year. The filing fee is $100. New Hampshire has no state income tax on earned income, making the annual report the primary compliance requirement.

When is it due?

The annual report is due April 1 each year.

How much does it cost?

The annual report fee is $100.

How to file

Filing steps

Go to NH QuickStart
Search for your LLC
File the annual report
Pay the $100 fee
Save confirmation

What happens if you miss the deadline?

Continued non-filing leads to administrative dissolution.

Do you need a registered agent in New Hampshire?

Yes. New Hampshire requires all LLCs to maintain a registered agent with a New Hampshire address.

Need a registered agent in New Hampshire?

A registered agent keeps your LLC in good standing and ensures you receive all legal and state documents on time.

Frequently asked questions

Does New Hampshire have income tax?

New Hampshire has no tax on earned income. There is a tax on interest and dividends (being phased out).

Can I file online?

Yes. File through the NH QuickStart portal.

What is the deadline?

April 1 each year for all New Hampshire LLCs.
